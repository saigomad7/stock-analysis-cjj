📱 화면별 기능
🏠 홈
  • 교집합 종목 — 기간1 기준과 순위 기준 두 결과에 모두 등장한 종목. 신뢰도 가장 높음.
  • 기준 선택 — 📊 기간1 기준(단기 가중) / 🏆 순위 기준(종합) 전환 가능.
  • 정렬 — 전체순위·기간총점수·긍정부정·종가목표·매수신호 기준 정렬.
📋 종목
  • 기간1/순위 기준 — 최종 추천 종목 상세 테이블.
  • 교집합 — 기간1기준 top30 AND 순위기준 top30 동시 해당 종목.
  • 합집합 — 기간1기준 top30 OR 순위기준 top30 해당 종목 전체.
  • 정렬 — 컬럼 기준 정렬 선택 가능.
  • 종목명·종목코드로 실시간 검색.
📊 시장
  • 하락 수준 추이 — 전체/단기추천/장기추천 종목 중 하락 비율(%). 80% 이상이면 과매도 구간.
  • 상승하강 평균 — 전체 종목의 일별 평균 등락률. -1.5% 이하면 급락 구간.
⚙️ 설정
  • API 서버 URL — run.sh가 출력하는 Cloudflare 주소. 재시작마다 업데이트 필요.
  • 기간별 가중치 — 스코어 계산 시 각 기간의 비중 (합계 1.0 권장).
  • 분석 기간 — 기간1~2: 일 단위, 기간3~5: 월 단위, 기간6: 년 단위.
  • 필터 파라미터 — para1~17. 0으로 설정 시 해당 조건 비활성.
🧠 분석 로직
STEP 1 — 종목 필터링
  • KRX 전체 ~2,700 종목에서 거래량 10만주 이상 종목만 추출.
  • 종가 범위(para15~16), 영업이익(para17), 외국인소진(para4) 조건으로 필터.
  • 상장 거래일 수(para14 = 80일 이상) 조건으로 신규상장 제외.
STEP 2 — 6개 기간 수익률 계산
  • 각 기간(기간1~6)의 초반 n일 평균 vs 최근 n일 평균을 비교한 변화율(%).
  • 기간1(15일)·기간2(30일): 단기 / 기간3(2개월)·기간4(3개월): 중기 / 기간5(6개월)·기간6(1년): 장기.
  • 가중치(기간1~4: 0.2, 기간5~6: 0.1)를 곱해 통합 스코어 산출.
STEP 3 — 기술적 지표
  • RSI — 14일 상대강도지수. 과매도(≤30) / 과매수(≥70) 신호 비율 계산.
  • 볼린저밴드 — 14일 이동평균 ± 2σ. 하단 이탈(매수신호) / 상단 이탈(매도신호) 횟수.
  • OBV — 거래량 기반 매수·매도 압력. 강한 매수(거래량 200만주↑) 따로 집계.
  • 선형회귀 — 최근 n일 종가에 LinearRegression 적용. 기울기 양수 = 우상향 추세.
STEP 4 — 최종 스코어 및 필터
  • 6기간 가중합 스코어를 0~100점으로 정규화. 순위(total_rank) 함께 산출.
  • 목표주가 있는 종목만 통과(네이버 재무 캐시 기준).
  • 긍정부정: 최근 60일 상승/하락일에 지수 가중치 적용한 차이값. 양수=상승 우세.
  • 기간1 가중 스코어 기준 결과 + 종합순위 기준 결과 두 벌 동시 산출. 교집합이 핵심.
📐 지표 해석
지표좋은 신호나쁜 신호
기간1~6양수 (상승 중)음수 (하락 중)
회귀선양수 (20일 추세 우상향)음수 (20일 추세 하락)
긍정부정높을수록 상승 우세음수 = 하락 우세

신호 n/43~4 = 매수 신호 강함0~1 = 신호 약함
RSIrsi_lo 높음 = 과매도 후 반등 가능rsi_up 높음 = 과매수
